## Comand line
### Osnove
Kalkulator, ki lahko izvaja aritmetične (+, -, *, %) in logične (and, or, not) operacije.

Osnovno računanje. Seštevanje, odštevanje, množenje, deljenje

Primeri:

```python
>>> 1+1
2
>>> 5-4
1
>>> 2*5
10
>>> 9/3
3.0
```
Napredno računanje. Potenciranje, ostanek

Primeri:
```python
>>> 4**2
16
>>> 2**10
1024
>>> 5%2
1
```
Oklepaji

Primeri:
```python
>>> 4 + 2*5
14
>>> (4+2) * 5
30
```
